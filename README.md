## 📋 What This Project Covers
This project tests whether Python Pandas
can completely replace Microsoft Excel.

## ✅ What Was Tested
1. 📊 All 15 Excel Chart Types
   - Bar, Column, Pie, Donut, Line
   - Area, Scatter, Bubble, Histogram
   - Box Plot, Heatmap, Stacked Bar
   - Radar, Waterfall, Combo

2. 🔢 All Excel Formulas
   - Math & Trig (SUM, ROUND, SQRT, LOG...)
   - Statistical (MAX, MIN, RANK, CORREL...)
   - Text (UPPER, LEN, LEFT, TRIM...)
   - Logical (IF, IFS, AND, OR, NOT...)
   - Lookup (VLOOKUP, INDEX, MATCH...)
   - Date (TODAY, YEAR, DATEDIF...)
   - Financial (PMT, FV, PV, NPV, IRR...)
   - Information (ISBLANK, ISNUMBER...)

3. 📋 All Data Options
   - Sort, Filter, Remove Duplicates
   - Data Validation, Pivot Table
   - Group & Subtotal, What If Analysis
   - Flash Fill, Text to Columns

4. 👁️ All View Options
   - Freeze Panes, Split View
   - Zoom, Page Layout, Gridlines

5. 🤖 Automate
   - Macros, Auto Fill, Auto Format
   - Auto Save, Scheduled Tasks
   - Batch Processing

6. ✍️ Draw
   - Shapes, Annotations
   - Freeform Drawing, Icon Sets

7. 📝 Review
   - Spell Check, Comments
   - Track Changes, Protect Sheet
   - Error Checking

8. 🗄️ SQLite Database
   - Created database from scratch
   - 50 students data
   - SQL queries

## 📊 Dataset

  History, Computer
